# *MIREA-PRACT*

+ Физика 1 семестр
+ Процедурное программирование

+ Математический анализ 1 семестр
### 2 семестр
+ Иностранный (английский) язык 2 семестр

+ Математический анализ 2 семестр

+ Русский язык и культура речи

+ Физика 2 семестр
### 3 семестр
+ Архитектура вычислительных машин и систем
